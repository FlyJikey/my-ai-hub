# Возможности каталога товаров

## Обзор

Система управления каталогом товаров с векторным поиском и интеграцией с чатом ИИ.

## Основные функции

### 1. Загрузка файлов

**Поддерживаемые форматы:**
- CSV
- Excel (.xlsx, .xls)
- JSON

**Два режима загрузки:**
- ✅ **Заменить существующую базу товаров** - полностью удаляет старые данные и загружает новые
- ⬜ **Добавить к существующим** - добавляет новые товары к текущим

**Процесс:**
1. Перетащите файл или нажмите для выбора
2. Выберите режим (замена/добавление)
3. Нажмите "Загрузить в базу"
4. Система покажет прогресс загрузки

### 2. Векторизация товаров

**Выбор модели для векторизации:**

Доступны 4 модели embedding:

**Бесплатные:**
- **Nomic Embed 1.5** (OpenRouter) - 768 измерений

**Платные (Polza.ai):**
- **OpenAI Embedding Small** (рекомендуется) - 1536 измерений, быстрая и качественная
- **OpenAI Embedding Large** - 3072 измерения, максимальная точность
- **OpenAI Ada 002** - 1536 измерений, проверенная модель

**Процесс векторизации:**
1. После загрузки файла появится блок "Требуется векторизация"
2. Выберите модель из выпадающего списка
3. Нажмите "Векторизовать ИИ"
4. Система обработает только невекторизованные товары
5. При завершении появится зелёная галочка "Векторизация завершена успешно!"

**Важно:**
- Векторизуются только товары без векторов (с `embedding = null`)
- Прогресс отображается в реальном времени
- Можно остановить процесс кнопкой "Остановить векторизацию"

### 3. Статусы готовности базы

**Зелёный статус:**
```
✓ База полностью готова к анализу
```
Все товары векторизованы, можно использовать в чате.

**Жёлтый статус:**
```
⚠ База не довекторизована, поиск будет неполным
```
Есть товары без векторов, требуется векторизация.

### 4. Интеграция с чатом ИИ

**Как использовать:**
1. Откройте "Чат ИИ"
2. Включите переключатель "База товаров" (иконка Database)
3. Задайте вопрос о товарах

**Что может ИИ:**
- Искать товары по описанию
- Сравнивать характеристики
- Рекомендовать товары
- Отвечать на вопросы о наличии и свойствах

**Как работает:**
1. Ваш запрос преобразуется в вектор
2. Система находит топ-20 релевантных товаров через векторный поиск
3. ИИ получает информацию о найденных товарах
4. ИИ отвечает на основе реальных данных из каталога

## Технические детали

### Структура данных товара

```javascript
{
  sku: "артикул",
  name: "название",
  category: "категория",
  description: "описание",
  price: 1000,
  attributes: { /* характеристики */ },
  raw_text: "полный текст для векторизации",
  embedding: [/* вектор 1536 или 3072 измерений */]
}
```

### API Endpoints

- `POST /api/catalog/upload` - загрузка файла
- `POST /api/catalog/vectorize` - векторизация товаров
- `GET /api/catalog/stats` - статистика каталога
- `POST /api/catalog/search` - векторный поиск
- `POST /api/catalog/chat` - чат с RAG по каталогу

### Ограничения

- **Timeout:** 60 секунд на операцию
- **Размер файла:** максимум 50MB
- **Batch размер загрузки:** 1000 товаров за раз
- **Batch размер векторизации:** 200 товаров за раз
- **Параллельность:** 10 одновременных батчей

### Оптимизация для больших каталогов

Если у вас более 3000 товаров:
1. Разбейте файл на части по 2000-3000 товаров
2. Загружайте частями с режимом "Добавить к существующим"
3. Векторизуйте после каждой загрузки

## Решение проблем

### Ошибка 504 (Timeout)

**Причина:** Операция занимает больше 60 секунд

**Решение:**
- Уменьшите размер файла
- Разбейте на несколько частей
- Проверьте скорость интернета

### Товары не находятся в чате

**Проверьте:**
1. Все ли товары векторизованы (зелёный статус)
2. Включён ли переключатель "База товаров" в чате
3. Есть ли в запросе ключевые слова из товаров

### Дубликаты товаров

**Причина:** Загрузка с режимом "Добавить" вместо "Заменить"

**Решение:**
1. Включите чекбокс "Заменить существующую базу товаров"
2. Загрузите файл заново

## Примеры использования

### Пример 1: Первая загрузка каталога

```
1. Загрузите CSV файл с 3000 товарами
2. Включите "Заменить существующую базу товаров"
3. Нажмите "Загрузить в базу"
4. Дождитесь завершения (прогресс 100%)
5. Выберите модель "OpenAI Embedding Small"
6. Нажмите "Векторизовать ИИ"
7. Дождитесь зелёной галочки
8. Откройте "Чат ИИ" и включите "База товаров"
9. Спросите: "Покажи все красные товары"
```

### Пример 2: Добавление новых товаров

```
1. Загрузите CSV с 500 новыми товарами
2. Отключите "Заменить существующую базу товаров"
3. Нажмите "Загрузить в базу"
4. Появится уведомление о 500 невекторизованных товарах
5. Выберите ту же модель, что использовали ранее
6. Нажмите "Векторизовать ИИ"
7. Новые товары станут доступны в чате
```

### Пример 3: Смена модели векторизации

```
⚠️ ВНИМАНИЕ: При смене модели нужно перевекторизовать ВСЕ товары!

1. Удалите текущую базу (загрузите пустой файл с заменой)
2. Загрузите каталог заново
3. Выберите новую модель (например, OpenAI Large)
4. Векторизуйте все товары
```

## Рекомендации

1. **Используйте "OpenAI Embedding Small"** - оптимальный баланс цены и качества
2. **Векторизуйте сразу после загрузки** - не откладывайте
3. **Проверяйте зелёный статус** перед использованием в чате
4. **Используйте режим "Заменить"** для обновления каталога
5. **Следите за логами** в консоли браузера (F12) при проблемах
